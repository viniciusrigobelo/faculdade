# 🦸 X-Men HQs – Loja Virtual

Projeto desenvolvido em **HTML, CSS e JavaScript**, com foco em **desenvolvimento front-end**, **organização de código** e **estética baseada nos quadrinhos dos X-Men**, utilizando cores e design inspirados no tema da franquia.

> Observação: Este projeto começou como um trabalho acadêmico em 2023, no ensino médio, e evoluiu até o formato atual, estruturado de forma profissional, pronto para compor meu portfólio.

---

## 📌 Sobre o Projeto

O **X-Men HQs** é uma loja virtual de quadrinhos de super-heróis, que mantém uma estética inspirada nos X-Men, com destaque para as cores **amarelo e vermelho** no layout.  

O projeto foi desenvolvido com o objetivo de aplicar conceitos fundamentais do desenvolvimento web, como:

- Estruturação de HTML semântica  
- Estilização avançada com CSS  
- Organização de código modular em arquivos separados (CSS e JS)  
- Planejamento para futuras funcionalidades em JavaScript  
- Manutenção de identidade visual coerente com o tema dos X-Men  

---

## 🎯 Funcionalidades

- Exibição de produtos em cards  
- Destaque visual de preços e promoções  
- Layout pensado para desktop (sem media queries)  
- Navegação por categorias (nav)  
- Formulário de login com validação básica de campos  
- Estrutura pronta para futuras interações em JavaScript  
- Layout inspirado nos X-Men, mantendo cores e estilo da franquia  

---

## 🧠 Organização e Funcionamento

O projeto mantém:

- **HTML semântico** para cada seção (header, nav, main, footer)  
- **CSS modularizado**, com arquivos separados para estilos globais, página home e login  
- **JavaScript** referenciado (arquivo `main.js`) pronto para futuras implementações  

O design busca um equilíbrio entre funcionalidade e estética, seguindo a identidade visual dos X-Men, tornando a interface agradável e coerente para o usuário.

## ▶️ Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/alekxnv/x-men-hqs.git
```
2. Abra o arquivo `index.html` no navegador.  
3. Explore as páginas e funcionalidades do layout.  
4. O arquivo `main.js` está incluído e pronto para receber scripts futuros.
